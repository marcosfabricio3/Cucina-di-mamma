# Cucina di Mamma

Aplicación web gastronómica inspirada en la cocina italiana tradicional.
Desarrollada como proyecto full-stack utilizando tecnologías web modernas y arquitectura simple cliente-servidor.

## Demo en producción
 https://cucina-demo.lovestoblog.com/index.html

## Descripción

Cucina di Mamma es un sitio web informativo con sistema de contacto persistente en base de datos.

El proyecto combina:

Frontend estático optimizado
Backend en PHP orientado a API
Persistencia en MySQL
Despliegue en hosting compartido (InfinityFree)
Incluye validación tanto del lado del cliente como del servidor, manejo de códigos HTTP y respuesta estructurada en JSON.

## Stack Tecnológico

### Frontend

- HTML5 semántico
- CSS3 modularizado
- JavaScript vanilla (Fetch API)
- Validación del lado cliente

### Backend

- PHP 8+
- PDO para conexión segura a base de datos
- Prepared Statements
- Manejo de excepciones
- Respuestas JSON estructuradas

### Base de datos

- MySQL
- Esquema relacional simple
- Tabla consultas
- Infraestructura
- InfinityFree (hosting compartido)
- phpMyAdmin
- FTP (FileZilla) para despliegue
